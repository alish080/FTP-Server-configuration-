# FTP-Server-configuration-
Configurating FTP Server in Cisco Packet Tracer.

This project shows how FTP Server is configured and how files are transferred between a client and a server using FTP command.

# Objective 
- Configure FTP server
- Connet client PCs to server
- Upload and Download files using FTP
----------------------------------------------------------------------------------------------------------------------

# Network Setup
- 2 PCs
- 1 Switch
- 1 Router
- 1 server
----------------------------------------------------------------------------------------------------------------------

# Ip Addressing
- Ftp Server: 10.0.0.2
- Clinet Server : 192.168.1.10
----------------------------------------------------------------------------------------------------------------------

# Tools used
- Cisco Packet Tracer
- Ftp server
- Command line interface
----------------------------------------------------------------------------------------------------------------------

# File include
- FTP Server Configuration PDF
- FTP Server Configuration Docx file
- FTP Server Configuration ".PKT"
